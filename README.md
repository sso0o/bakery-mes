# 🍞 MES System for Bakery Factory

제과 공장의 생산 이력 추적 및 작업지시 관리를 위한 Java 기반 MES 시스템

## 주요 기능

- 제품 등록 / 공정 등록 / 상태 등록
- 작업지시 생성 및 LOT 번호 자동 생성
- 자재 입고 및 재고 관리
- 제품 공정별 자재 소모 관리
- 공정별 생산 실적 등록

## 기술 스택

- **Backend**: Java, Spring Boot
- **Frontend**: React
- **Database**: MySQL
- **Build Tool**: Gradle
